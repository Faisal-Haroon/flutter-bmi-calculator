# Flutter BMI Calculator

A simple BMI (Body Mass Index) calculator app built using Flutter.  
This project was created for practice to understand Flutter basics and input validation.

## Features
- Enter weight (kg) and height (cm)
- Calculates BMI using standard formula
- Shows BMI value up to 2 decimal places
- Displays BMI category:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Input validation:
  - Empty fields not allowed
  - Zero or negative values not allowed
  - Safe number parsing (no app crash)
- Keyboard overflow handled using scroll view
